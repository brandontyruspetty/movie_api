myNoirMovies

This application will provide users with the ability to access information about movies and directors within the Film Noir genre as well as create a profile for themselves and create lists of their favorite movies.

Request URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None None A JSON object holding data about ALL the Movies
Return data about a single movies by title to the user /movies/[MovieName] GET :MovieName None A JSON object holdiong data about a single movie containing a description, genre, director and image URL
Return data about a genre (description) by name (e.g., "Contemporary Noir") /movie/[Genre]/[GenreName] GET :GenreName None A JSON object holding data (description) about a single Genre
Return data about a director (bio, birth year, death year) by name /movies/[Director] GET :Director None A JSON object holding data (bio, birth year, death year) about a single Director

Allow users to update their user info (username, password, email, date of birth) /users/[UserName] PUT :UserName None A text message indicating the user and information that was updated
Allow users to add a movie to their list of favorites /users/[id]/[MovieName] Post :id, :MovieName A JSON object holding data about the user and the movie added, structured like this { "name" : John/Jane Doe, "moviename" : "noir-tastic" } A JSON object holding data about the user and the movie added, structured like this { "name" : John/Jane Doe, "moviename" : "noir-tastic" }

Allow existing users to deregister /users/[UserName]/[ID] DELETE :UserName, :id None A text message indicating whether the user was successfully removed
