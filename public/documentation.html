myNoirMovies

This application will provide users with the ability to access information about movies and directors within the Film Noir genre as well as create a profile for themselves and create lists of their favorite movies.

Request URL HTTP Method Response Body Data Format
Return a list of ALL movies to the user /movies GET A JSON object holding data about ALL the Movies
Return data about a single movies by title to the user /movies/[Name] GET A JSON object holdiong data about a single movie containing a description, genre, director and image URL
